FubuRake

FubuRake is a standard Ruby library to quickly stand up a fully functional, cross platform rake script for a .Net codebase based on Fubu project standards. FubuRake also provides standard integration with LINK TO PROJECT 'Ripple' for package management and FubuDocs for documentation generation.
